Patchwork

Patchwork is a PHP library that makes it possible to redefine user-defined functions and methods
at runtime, loosely replicating the functionality of runkit_function_redefine in pure PHP 5.4 code,
which, among other things, enables you to replace static and private methods with test doubles.

function size($x)
{
    return count($x);
}

size([1, 2]); # => 2

1. Define a function

use Patchwork as p;

p\redefine('size', function($x) {
    $u = str_repeat('u', p\relay();
    return 'h' . $u . 'ge';
});

size([1, 2]); # => "huuge"

2. Replace its definition

p\restoreAll();

size([1, 2]); # => 2

# Note that all of these steps
# occur at the same runtime!

3. Undo the redefinition

We make the assumption of a responsible developer, of one that will continue writing maintainable code despite the hurdle of testability being lowered, meaning that a major motivator for maintainablity is eliminated.

As of now, it is strongly recommended to limit the use of Patchwork to testing and development environments only.
